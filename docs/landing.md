# OPC Template — One Person Company

**OPC Template** 是一个面向个人开发者的 AI 原生软件工厂模板。
它把 **设计 → Schema → 生成 → 运行 → 自愈** 连成一条可执行的生产线。

## Key Features

- Schema-first: 结构先于代码
- AI Self-Healing: The Loop 自动修复
- Cross-stack: 小程序前端 + Go/Gin 后端 + PostgreSQL
- Forkable: 一键 Fork 可用

## Quick Start

```bash
git clone https://github.com/yourname/opc-template
cd opc-template
make init
make design
make generate
make run
```
